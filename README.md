# Cursor Parking Lot

- A Chrome Extension that allows you to "park" and lock your mouse cursor inside a specific area on the screen.

# Features

- Locks the mouse cursor inside a draggable box using the Pointer Lock API.

- Saves the parking spot's location between page reloads.

# Installation

- Save the extension files (manifest.json, content.js, style.css) to a folder.

- Open Chrome and go to chrome://extensions.

- Enable "Developer mode" (top right).

- Click "Load unpacked" and select your folder.

# Usage

- Click and drag the box to move it.

- Click the "P" icon to park (lock) the cursor.

- Press ESC to release the cursor.
